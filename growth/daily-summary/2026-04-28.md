## Session 5: Keybinding Conflict Detection

Implemented warning-only keybinding conflict detection in Settings.

### Goal

Warn when two commands share the same effective shortcut while still allowing the user to save/edit bindings. This keeps the feature helpful without making the keybinding editor brittle.

### Conflict Utility

Created `src/lib/keybinding-conflicts.ts`:

- `normalizeHotkey()`
  - normalizes modifier order
  - lowercases single-character keys
  - treats `Ctrl`, `Control`, `Cmd`, and `Command` as `Mod`
  - normalizes aliases like `Space` and `Escape`
- `findKeybindingConflicts()`
  - groups commands by normalized effective hotkey
  - returns duplicate groups with command IDs and labels

Added `src/lib/keybinding-conflicts.test.ts`:

- modifier ordering and key casing
- platform modifier aliases
- duplicate detection
- blank binding handling

### Settings UI
